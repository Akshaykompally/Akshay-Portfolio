My Projects

Each project below is documented as a case study: the problem I was trying to solve, how I approached it, what actually shipped, and what I'd do differently.

PROJECT 01

Interview Tracker

Full-stack web application for interview performance analytics

Python Flask MySQL Chart.js Flask-Login Bcrypt
GitHub

THE PROBLEM

Preparing for interviews can become disorganized, with notes scattered across multiple platforms and no clear way to track progress. I needed a centralized system to record interview experiences, monitor performance, and identify recurring weak areas. Existing solutions lacked personalization and analytics focused on interview preparation.

MY APPROACH

I developed a full-stack web application using Flask, MySQL, HTML, CSS, and JavaScript. The system allows users to log interview details, track outcomes, and manage records through CRUD operations. I integrated Flask-Login and Bcrypt for secure authentication and used Chart.js to visualize interview performance data.

OUTCOME

The application provided a structured way to manage interview preparation and track growth over time. Users could analyze performance trends through charts and identify strengths and weaknesses. The dashboard made interview data easier to understand and helped improve preparation efficiency.

WHAT I'D CHANGE

I would add role-specific interview roadmaps and AI-generated preparation recommendations. I would also implement email reminders, company-specific preparation checklists, and cloud deployment to improve scalability and accessibility.

PROJECT 02

AI Image Generation App

ClipDrop-powered image generation with Flask backend

Python Flask MySQL ClipDrop API HTML CSS
GitHub

THE PROBLEM

Creating AI-generated images often requires using multiple tools and platforms, making the process less convenient for users. I wanted to provide a simple interface where users could generate images from text prompts without dealing with complex AI workflows.

MY APPROACH

I built a Flask-based web application integrated with the ClipDrop API. The system processes user prompts, communicates with the AI service, and displays generated images. Authentication, session management, and MySQL storage were implemented to provide a secure and personalized experience.

OUTCOME

The application successfully generated images based on user prompts through seamless API integration. Users could securely access the platform and manage their generated content. The project improved my understanding of API integration, backend workflows, and AI-powered applications.

WHAT I'D CHANGE

I would support multiple AI providers instead of relying on a single API. Adding image history, prompt optimization, image editing features, and cloud storage would improve usability and make the application more scalable.I'd also look at R2 or S3 for storing generated images rather than the local filesystem.

PROJECT 03

Facial Age Estimation

CNN-based deep learning model for age group classification

Python Flask CNN Numpy Matplotlib HTML/CSS
GitHub

THE PROBLEM

Estimating age from facial images is a challenging computer vision problem due to variations in lighting, image quality, facial expressions, and age-related features. The goal was to develop a model capable of accurately classifying age groups from images.

MY APPROACH

I developed a Convolutional Neural Network (CNN) model and implemented preprocessing techniques such as resizing, normalization, and feature extraction. Facial images were processed through a structured pipeline to improve consistency and model performance. NumPy and Matplotlib were used for data handling and visualization.

OUTCOME

The model achieved approximately 98% classification accuracy across predefined age groups. Performance metrics such as Precision, Recall, and F1-Score demonstrated strong model effectiveness. The project enhanced my knowledge of deep learning, image processing, and model evaluation.

WHAT I'D CHANGE

I would train the model on a larger and more diverse dataset to improve generalization. I would also experiment with transfer learning models such as ResNet and EfficientNet and deploy the solution as a real-time web application for practical use.
